# Contributing

**This repository is in active development and is not currently accepting pull requests.**

The repo is public so that GitHub Actions runs against the format library and CLI tools without consuming the private-repo Actions quota. The visibility does not signal that the project is open to outside contribution yet — the design is still settling, the codebase is moving fast, and there is no review capacity for outside PRs.

## What you can do

- **File issues** — bug reports against `alo_dump` / `alo_roundtrip` are welcome (especially if you find a vanilla `.alo` or `.ala` file that doesn't round-trip cleanly). Issues are limited to repo collaborators during development; if you can't open one, feel free to reach out via GitHub.
- **Fork freely** — the MIT license applies. If you want to experiment, fork is the right tool; just don't expect those changes to flow back upstream during this phase.

## When PRs will open up

Once the v1.0 exporter is feature-complete (EaW + FoC export, including animation and visibility tracks), this file will be updated with real contribution guidelines: coding conventions, test expectations, the format-knowledge bar for chunk-format changes, etc.

Until then: thanks for your interest, and please be patient.
